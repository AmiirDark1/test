# 🚀 ماینکرفت سرور + پنل مدیریت

اینجا یه **سرور ماینکرفت** به همراه **پنل مدیریت** داریم که با داکر اجرا میشه.

---

## 🧰 نیازمندی‌ها (چی باید نصب باشه؟)

| نیازمندی | لینک دانلود |
|---------|------------|
| **Docker Desktop** | https://www.docker.com/products/docker-desktop/ |
| **Git** (اختیاری) | https://git-scm.com/ |

> فقط کافیه Docker Desktop رو نصب کنی. بقیه چیزا خودکار نصب میشه.

---

## ⚙️ روش راه‌اندازی

### ویندوز (PowerShell)

```powershell
# ۱- فایل setup.ps1 رو اجرا کن
# روش اول: روی فایل setup.ps1 کلیک راست کن → Run with PowerShell
# روش دوم: توی ترمینال PowerShell برو کنار فایل و بنویس:
.\setup.ps1
```

### لینوکس / مک (Bash)

```bash
# تو ترمینال برو تو پوشه پروژه و اینو بزن:
bash setup.sh
```

### دستی (اگه دوست داری خودت تک‌تک انجامشون بدی)

```bash
# ۱- نصب پکیج‌های npm
cd mc-panel && npm install

# ۲- اجرای کانتینرها با داکر
cd ..
docker compose up -d --build
```

## 📌 دستورات کاربردی

```bash
# نمایش لاگ‌ها
docker compose logs -f

# متوقف کردن همه سرویس‌ها
docker compose down

# ریستارت کردن سرویس‌ها
docker compose restart

# وضعیت کانتینرها
docker compose ps

# دسترسی به کنسول ماینکرفت (RCON)
docker exec -it minecraft rcon-cli
```

## ❓ سوالات متداول

### ۱. ارور `Docker not found` میده، چیکار کنم؟
> برو از سایت docker.com Docker Desktop رو دانلود و نصب کن. بعدش دوباره اجرا کن.

### ۲. پورت 3000 گرفته است؟
> برو فایل `docker-compose.yml` رو باز کن و خط `"3000:3000"` رو به یه پورت دیگه مثلاً `"4000:3000"` تغییر بده. بعد پنل توی `http://localhost:4000` باز میشه.

### ۳. چجوری رمز RCON رو عوض کنم؟
> توی فایل `docker-compose.yml` مقدار `RCON_PASSWORD: "minecraft"` رو به هر چی میخوای تغییر بده.

### ۴. سرور ماینکرفت استارت نمیشه؟
```bash
docker logs minecraft
```
> ببین لاگ چی میگه. معمولاً اولین بار باید EULA رو قبول کنه که خودکار تو فایل انجام شده.

### ۵. چجوری پلاگین اضافه کنم؟
> فایل `.jar` پلاگین رو بنداز تو `data/plugins/` و بعد سرور رو ریستارت کن:
```bash
docker compose restart minecraft
```

- این پروژه برای **تست و استفاده شخصی/لوکال** ساخته شده
