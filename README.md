# 🚀 ماینکرفت سرور + پنل مدیریت

اینجا یه **سرور ماینکرفت** به همراه **پنل مدیریت** داریم که با داکر اجرا میشه.

---

## ☝️ نصب با یه خط دستور (فقط لینوکس)

```bash
bash <(curl -Ls https://raw.githubusercontent.com/AmiirDark1/test/master/install.sh)
```

فقط کافیه این خط رو تو ترمینال کپی کنی و بزنی اینتر. اسکریپت خودش همه چیز رو انجام میده: چک میکنه Docker هست یا نه (اگه نباشه نصبش میکنه)، فایل‌ها رو دانلود میکنه، و سرور + پنل رو بالا میاره.

---

## 📄 اگه فایل‌ها رو از قبل داری

```bash
bash setup.sh
```

---

## 🌐 بعد از اجرا

| سرویس | آدرس |
|-------|------|
| **پنل مدیریت** | http://localhost:3000 |
| **سرور ماینکرفت** | `localhost:25565` |
| **RCON** | پورت `25575` / رمز: `minecraft` |

---

## 📂 ساختار پوشه‌ها

```
📁 پروژه
├── 📄 install.sh            # نصب‌کننده یه‌خطی (لینوکس)
├── 📄 setup.sh              # فایل ستاپ (لینوکس)
├── 📄 docker-compose.yml    # تنظیمات داکر (سرور + پنل)
├── 📁 data/                 # دیتای سرور ماینکرفت
│   ├── 📁 world/            # جهان اصلی
│   ├── 📁 plugins/          # پلاگین‌ها
│   └── 📁 logs/             # لاگ‌ها
└── 📁 mc-panel/             # سورس پنل مدیریت
```

---

## 📌 دستورات کاربردی

```bash
# نمایش لاگ‌ها
docker compose logs -f

# متوقف کردن همه سرویس‌ها
docker compose down

# ریستارت کردن سرویس‌ها
docker compose restart

# وضعیت کانتینرها
docker compose ps

# دسترسی به کنسول ماینکرفت (RCON)
docker exec -it minecraft rcon-cli
```

## ❓ سوالات متداول

### ۱. ارور Docker میده، چیکار کنم؟
برو از سایت docker.com Docker رو نصب کن. لینوکس:
```bash
curl -fsSL https://get.docker.com | sh
```

### ۲. پورت 3000 گرفته است؟
برو فایل `docker-compose.yml` رو باز کن و خط `"3000:3000"` رو به یه پورت دیگه مثل `"4000:3000"` تغییر بده.

### ۳. چجوری رمز RCON رو عوض کنم؟
توی `docker-compose.yml` مقدار `RCON_PASSWORD: "minecraft"` رو عوض کن.

### ۴. سرور ماینکرفت استارت نمیشه؟
```bash
docker logs minecraft
```
ببین تو لاگ چی نوشته.

### ۵. چجوری پلاگین اضافه کنم؟
فایل `.jar` پلاگین رو بنداز تو `data/plugins/` و بعد سرور رو ریستارت کن:
```bash
docker compose restart minecraft
```
